# ai-tinkerers-indiegrow-demo

The code of the Community Marketing agent we will be showing.

## Setup

```
bun install
```

## Config

You need the following constants in your .env:

```
OPENAI_API_KEY=
TAVILY_API_KEY=
```

## Execute

```
bun src/toolCallingAgent.ts URL_OF_YOUR_APP_OR_WEBSITE
```
